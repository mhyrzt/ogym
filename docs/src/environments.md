# Environments

OGym currently supports the following environment modules:

- **`env/control`:** Classic control benchmarks such as CartPole and Pendulum.
- **`env/toy_text`:** Discrete environments including FrozenLake, Taxi,
  Blackjack, and CliffWalking.
- **`env/rapier`:** Physics-based environments powered by the
  [Rapier](https://rapier.rs/) engine.
- **`env/mujoco`:** High-fidelity physics environments using MuJoCo.
