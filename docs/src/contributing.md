# Contributing

Contributions from the community are welcome.

- **Bug reports and feature requests:** Open an issue on
  [GitHub](https://github.com/mhyrzt/ogym/issues).
- **Pull requests:** Submit pull requests for bug fixes or new environments.
  For major architecture changes, open an issue first so the approach can be
  discussed.
